>  정릉3동주민자치회의 **「정릉3동 마을시간은행 활성화 사업」** 은 생활권 동네 단위로 이뤄져있으며,  
>  마을시간은행은 서로 돕고 챙겨주는 복지 품앗이 활동으로 정릉3동의 브랜드 사업이다.

## ✏️  버전 기록

## **v 1.2.0**
`Released: 2024.08.23`
> 8월 20일 주민센터 시연 후 받은 피드백을 반영하여 개선했습니다.

### 🛠️ 변경 사항
- **게시글 정보 표시 개선** : 도움 요청 글에서 요일 옆에 시작 시간이 함께 표시되어, 지원자들이 보다 명확한 정보를 확인할 수 있습니다.

### 🚀 새로운 기능
- **게시글 수정/삭제** : `모집중` 상태인 게시물에 대해서 게시물을 수정하거나 삭제할 수 있습니다.

## **v 1.1.1**
`Released: 2024.08.16`

### 🛠️ [안드로이드 웹앱] 버그 수정 및 개선 사항
- **채팅 기능 오류 수정** : 일부 안드로이드 기기에서 발생한 '채팅 보내기 버튼 사라짐' 문제를 해결했습니다.
- **디스플레이 모드 개선** : Fullscreen 모드에서 화면이 잘려 터치가 불편했던 문제를 해결하기 위해 디스플레이 모드를 Standalone으로 변경했습니다. 이제 보다 안정적이고 편리한 환경에서 앱을 사용할 수 있습니다.
- **안드로이드 웹앱 지원 범위 변경** : 안드로이드 웹앱의 지원 OS 범위를 변경하여 이제 8.0부터 14.0까지 지원합니다.

## **v 1.1.0**
`Released: 2024.06.04`
> 5월 14일 주민센터 시연 후 받은 피드백을 반영하여 기능을 추가했습니다.

### 🚀 새로운 기능
- **게시글 검색** : 현재  `모집중` 상태인 게시글을 검색할 수 있습니다. 카테고리별 검색도 가능합니다.
- **알림** : 새로운 게시글이 등록되었을 때, 내 게시글에 도움을 주겠다고 신청을 한 사람이 있을 때 알림을 발송해 드립니다.
- **카테고리** : 게시글에 도움 카테고리를 지정해 보고싶은 게시글만 모아 볼 수 있습니다.

## **v 1.0.0** 
`Released: 2024.05.14`
> 개발기간 : 2024.02 ~ 2024.05 (4개월)

### 🎉 주요 기능
- **사용자 친화적 UI/UX** : 정릉 3동 지역의 주 사용자층인 40대 후반부터 70대 초반을 고려하여 직관적이고 사용하기 쉬운 인터페이스를 제공합니다.
- **게시판** : 동네 이웃이 필요로 하는 도움글을 한 눈에 볼 수 있습니다. 둘러보며 내가 도울 수 있는 게시글에 지원해보세요!
- **글쓰기** : 누군가의 도움이 필요할 때 게시글을 작성해 사람을 모을 수 있습니다. 제목, 시작 시간, 장소, 소요 시간을 포함하여 작성해주세요.
- **채팅** : 1:1 채팅 또는 그룹 채팅으로 내 게시글에 지원한 사람들과 대화할 수 있습니다. 약속 장소와 시간을 구체적으로 이야기 나누어보세요.
